# 🚀 Real-Time Collaborative Code Sync System

A real-time collaborative code editor where multiple users can join a room and code together simultaneously.

## 🛠 Tech Stack
- React.js
- Node.js
- Express.js
- Socket.io
- CodeMirror

## ✨ Features
- Real-time code synchronization
- Multi-user collaboration
- Room-based sessions
- Syntax highlighting
- Join/leave notifications

## ⚙️ Setup

```bash
npm install
npm run server:dev
npm start
